PROJECT NOBRAND

[현재 인스턴스 중지 상태]

SSG Nobrand 웹사이트의 클론 코딩 프로젝트

사용 스택
주요 업무

  • 전반적인 데이터베이스 설계

  • 백엔드와 프론트엔드의 통신 설계

  • 로그인과 회원가입 페이지 구현

  • 기타 백엔드 관련 뷰와 리포지토리,서비스 설계

  • SHA-256을 활용한 저장 데이터 암호화와 복호화

  • AWS의 EC2를 활용한 웹페이지 배포

기술스택

Language

Java

  • Spring framework를 이용한 Spring Boot Project

  • JSP를 이용한 Maven Project

  • Apache Tomcat을 이용한 서버 구축

  • JDBC를 활용하여 Oracle, Mysql과의 통신

  • Servlet을 이용한 통신환경 구축

Javascript

  • API를 이용한 공공데이터 활용

  • Axios를 활용하는 비동기 통신

  • Jquery의 Datepicker를 활용하는 날짜 선택 및 날짜 데이터 반환

HTML

  • 웹 페이지 구현 가능

CSS

  • 페이지 스타일링 가능

Database

Oracle

  • Cmd를 활용하여 Database Schema 생성 및 user 생성, 권한 부여

  • SQLDeveloper를 활용하여 Database 개발환경 구현 및 데이터 실시간 확인

  • PK와 FK, Join을 활용하여 테이블 간의 관계 생성 및 데이터 활용 가능

MySQL

  • Cmd를 활용하여 Database Schema 생성 및 user 생성, 권한 부여

  • Mysql WorkBench를 활용하여 Database 개발환경 구현 및 데이터 실시간 확인

  • PK와 FK, Join을 활용하여 테이블 간의 관계 생성 및 데이터 활용 가능

  • WINDOW 환경과 AWS의 ubuntu 환경에서의 배포 경험

Server

apache

  • 기본적인 라우터 서버 구현 가능

  • Tomcat과 sevlet을 활용하여 서버 구현 가능

Apache_tomcat

  • Tomcat 9.0, Tomcat 10.0 사용 경험

  • WINDOW 환경과 AWS의 ubuntu 환경에서의 배포 경험

Deployment

AWS

  • EC2와 ROUTER53를 활용한 배포 경험

  • 가비아를 통한 개인 도메인 구매 및 해당 도메인에 호스팅 영역 연결

  • AWS Certificate를 사용한 SSL인증서 발급 가능

Ubuntu

  • AWS ubuntu 환경에서의 Tomcat9,Tomcat10 설치 및 활용 경험

Modules

CKeditor

  • CKeditor를 활용한 이미지 업로드 및 이미지 불러오기 가능

  • 기본적인 게시판 기능 구현 경험

Three.js

  • 랜덤 point와, point를 잇는 lines를 이용한 간단한 3D 파티클 구현 경험

Thymeleaf
